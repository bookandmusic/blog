GitBook 是一个基于Node.js的命令行工具，可以使用Markdown来制作电子书，并利用Git/Github发布。

支持输出静态站点、PDF、eBook、HTML网页等格式。

安装 GitBook 需要 Node 环境，具体怎么安装 Node 这里就不多说了。
