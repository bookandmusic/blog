高阶函数

描述

当一个函数 接收 另一个函数作为参数， 它就是 高阶函数

示例

内置函数

map

语法

描述

将 可迭代对象 中的每个值 按照 第一个参数( 函数)的 规则 依次 处理， 得到一个 新的 迭代器 对象

实例

reduce

语法

描述

将可迭代对象中所有的元素，按照第一个 函数的规则， 连续操作，得到一个 具体的数据

实例

filter

语法

描述

按照第一个 函数的 返回值 对 可迭代对象 中 所有的数据 进行 过滤， 得到一个 包含过滤后结果的 迭代器对象

实例

sorted

语法

描述

将可迭代对象 按照某种规则 排序，升序和降序均可。
